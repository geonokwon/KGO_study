# HTML의 기본 구성 요소
- 태그(<>)사이에 태그명의 넣는 형태
    ```html
    <태그명 속성명="속성값">
    ```
    
- 문법 -콘텐츠가 있는 문법( 시작태그 와 종료태그 를 가진다.)
    ```html
    <title>My First Web Fage!</title>
    ```
    
- 문법 -콘텐츠가 없는 문법( 빈 태그)
    ```html
    <br> 
    ```
    
- 주석
    ```html
    <!--이렇게 메모를 남길용도로 사용할수 있다.-->
    <!--대신 소스보기로 볼수있기때문에 보안상 중요한내용을 넣으면 안된다-->
    ```
